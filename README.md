# AWS CDK TYPESCRIPT

## Running Examples

To run a Typescript example, execute the following:

```sh
$ npm install -g aws-cdk
$ cd typescript/EXAMPLE_DIRECTORY
$ npm install
$ cdk deploy
```

Then, to dispose of the stack/s afterwards

```
$ cdk destroy
```
